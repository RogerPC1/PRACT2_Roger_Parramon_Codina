# PRACT2_Roger_Parramon_Codina
Codi de la Pràctica 2 de Tipologia i cicle de vida de les dades, consistent en analitzar les dades sobre criptomonedes provinents del web scraping de Yahoo Finances, amb l'objectiu de predir si certa criptomoneda tendirà a pujar o a baixar de preu.

# Integrants del grup:
Roger Parramon Codina

# Arxius del repositori:
Aquest mateix README.md, amb una breu descripció de la Pràctica 2 i el contingut d'aquest repository de GitHub.

La carpeta "/source", amb el codi d'R generat per tractar i analitzar les dades.

La carpeta "/datasets", que contè tant el fitxer amb les dades originals (provinent de la pràctica 1) com el fitxer resultant de la integració i neteja, emprat per a l'anàlisi; ambdós en format CSV.
